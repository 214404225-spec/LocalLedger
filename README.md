# **基于Android的本地记账的mini-APP**

## **核心价值**
本项目旨在解决现有商业记账 App 的两大痛点：  
- **强制联网/注册账号**：牺牲用户隐私，将消费数据上传至云端  
- **外币记账体验差**：无网环境下无法记录外币消费  

「本地隐私记账簿」通过 **离线优先 + 本地缓存汇率** 的设计，实现：  
- **完全离线可用**（无网仍可记外币账）  
- **零隐私泄露**（数据仅存本地，无任何网络上传）  
- **即用即走**（无需注册，打开即记账）

## **功能清单**
| 功能 | 说明 |
|------|------|
| **快速记账** | 输入金额、币种（CNY/USD/EUR/JPY/GBP）、类别、备注 |
| **智能汇率缓存** | 首次记外币账时联网获取汇率，24 小时内离线复用 |
| **编辑/删除** | 长按编辑、左滑删除，操作直观 |
| **统计图表** | 按类别（饼图）、按日（柱状图）可视化消费分布 |
| **隐私保障** | 仅申请 `INTERNET` 权限，无云同步、无广告 |
